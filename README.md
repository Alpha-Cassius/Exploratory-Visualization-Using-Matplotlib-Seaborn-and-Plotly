
# 🛒 Retail EDA & Visualization
An exploratory analysis of synthetic retail data using **Matplotlib**, **Seaborn**, and **Plotly** to identify consumer spending habits. This project demonstrates how to select the optimal chart type—bar, static scatter, or interactive scatter—based on specific analytical questions. Each visualization includes automated data aggregation and brief observational insights for a complete data storytelling workflow.
